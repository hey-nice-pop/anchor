# Discord Bot
wikipedia,weather,traffic...
## 必要
- discord.py
- wikipedia-api
```bash
pip install wikipedia
```
- requests
```bash
pip install requests
```
- beautifulsoup4
```bash
pip install beautifulsoup4
```
- geocoder
```bash
pip install geocoder
```
- python-dotenv
```bash
pip install python-dotenv
```
- .env
```.env
BOT_TOKEN = 'YOURTOKEN' # bottoken
```

- Wikipedia検索
- 最新ニュース取得
- 天気予報
